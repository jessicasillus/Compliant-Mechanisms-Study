Engineering Flow

How The Pieces Fit Together

The project is built as a connected pipeline instead of a disconnected set of files: analytical modeling, nonlinear FEA, data export, and interactive comparison all feed into one final discrepancy study.

01

Analytical PRBM

The one-spring model uses thickness-based torsional stiffness scaling and a cable moment arm to estimate joint bending and overall finger curl.

02

Distributed Comparison

The 20-spring reference keeps the same mechanism intent but spreads compliance more smoothly, helping isolate the effect of hinge lumping.

03

FreeCAD + CalculiX

The FEA model resolves the compliant joint volume directly and exports stage-by-stage joint and segment data for downstream comparison.

04

Interactive Comparison

The browser and desktop GUIs plot geometry, tip response, and joint-level discrepancy so the PRBM-versus-FEA gap can be interpreted visually.

Finger_PRBM.py

Thickness-scaled stiffness

def joint_torsional_stiffnesses_nmm_per_rad(
    joint_thicknesses_mm=JOINT_THICKNESSES_MM,
):
    return tuple(
        NOMINAL_JOINT_STIFFNESS_NMM_PER_RAD
        * (float(joint_thickness_mm) / NOMINAL_JOINT_THICKNESS_MM) ** 3
        for joint_thickness_mm in joint_thicknesses_mm
    )

Finger_Sim_FreeCAD.py

Matched FEA geometry build

def build_finger_shape():
    solids = [add_box(0.0, BASE_BLOCK_LENGTH_MM, SEGMENT_HEIGHT_MM)]
    joint_targets = []
    cursor_x = BASE_BLOCK_LENGTH_MM
    for joint_index, joint_thickness_mm in enumerate(
        JOINT_THICKNESSES_MM, start=1
    ):
        solids.append(add_box(cursor_x, COMPLIANT_SEGMENT_LENGTH_MM, joint_thickness_mm))
        solids.append(add_box(cursor_x + COMPLIANT_SEGMENT_LENGTH_MM,
                              RIGID_SEGMENT_LENGTH_MM, SEGMENT_HEIGHT_MM))

Local Workflow

Run the project locally

GitHub Pages is the easiest way to review the comparison, but the full project also includes the desktop app, the analytical scripts, and the FreeCAD/CalculiX source for regeneration work. 

python -m pip install -r finger_discrepancy_app/requirements.txt
python finger_discrepancy_app/Finger_Discrepancy_App.py

python finger_sim_freecad/Finger_PRBM.py --plot-schedule
python finger_sim_freecad/Finger_Sim_FreeCAD.py --help
python finger_sim_freecad/Finger_Sim_20_Springs.py

The large Finger_Sim_VM2_FEM.FCStd document is intentionally not committed because it is too large for a normal GitHub upload, but the generated staged CSV and the main solver scripts are included here.
